# ☁️ Day 54 — Introduction to Cloud for Data Science

## 🎯 Goal
Understand basic cloud concepts used in data science and machine learning.

---

## 📌 What is Cloud Computing?

Cloud computing means:
- storing data
- running applications
- using servers

through the internet instead of local computers.

---

## 🌐 Popular Cloud Platforms

- AWS (Amazon Web Services)
- Microsoft Azure
- Google Cloud Platform (GCP)

---

## 📊 Why Cloud is Important

Data scientists use cloud for:
- large datasets
- model training
- deployment
- storage
- collaboration

---

## 🧠 Common Cloud Services

| Service | Purpose |
|---------|---------|
| Storage | Save files/data |
| Compute | Run code/models |
| Database | Store structured data |
| ML Services | Train ML models |

---

## 💻 Example Use Cases

- Store datasets in cloud storage
- Deploy ML model as API
- Run notebooks online
- Create dashboards
