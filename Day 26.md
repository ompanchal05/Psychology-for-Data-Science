# 🤖 Day 26 — Behavioral Machine Learning Model
(Psychology + Data Science)

> “A model predicts behavior — a great data scientist explains it.”

---

## 📍 Overview

This stage focuses on building a **Machine Learning model** to predict user churn using **behavioral and psychological features**.

The goal is not only to achieve good accuracy, but also to:

- Understand **why users churn**
- Interpret model results using **psychology concepts**
- Connect **data → behavior → insights**

---

## 🎯 Objective

- Train a churn prediction model  
- Evaluate model performance  
- Identify important features  
- Explain results using psychology  

---

## 🧠 Behavioral Thinking Approach

Instead of treating features as numbers, we interpret them as:

| Feature | Psychological Meaning |
|--------|----------------------|
| tenure | Habit strength |
| support_calls | Frustration level |
| monthly_charges | Perceived cost/value |
| usage_frequency | Engagement level |

---

## 📊 Dataset

Example dataset: Customer Churn Dataset

Target variable:
- `churn` → (0 = stay, 1 = leave)

---

## 🔧 Model Used

### 1️⃣ Logistic Regression
- Simple
- Interpretable
- Good baseline

### 2️⃣ Random Forest
- Handles complex patterns
- Provides feature importance

---

## 🔀 Train-Test Split

- Training Data → 80%  
- Testing Data → 20%  

---

## 📈 Evaluation Metrics

- Accuracy  
- Precision  
- Recall  
- Confusion Matrix  

---

## 🔍 Feature Importance

Feature importance helps identify which variables most influence churn.

Example:

- High importance of `tenure`
- High importance of `support_calls`
- High importance of `monthly_charges`

---

## 🧠 Behavioral Insights

### Insight 1:
Users with low tenure churn more  
→ Weak habit formation  

---

### Insight 2:
Users with high support calls churn more  
→ Frustration leads to disengagement  

---

### Insight 3:
High monthly charges increase churn risk  
→ Low perceived value  

---

### Insight 4:
Frequent usage reduces churn  
→ Strong engagement and habit loop  

---

## 🚨 Key Learning

- Machine learning gives predictions  
- Psychology gives explanations  

Both together create powerful insights.
