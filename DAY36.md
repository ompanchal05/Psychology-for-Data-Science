# 🎤 Day 36 — Interview Practice (Recommendation Systems)

## 🎯 Goal
Prepare for interview questions related to recommendation systems.

---

## 📊 Key Questions

1. What is a recommendation system?  
2. Difference between collaborative and content-based filtering  
3. What is the cold start problem?  
4. How do you improve recommendations?  
5. What metrics are used?  

---

## 🧠 Important Concepts

- User similarity  
- Item similarity  
- Hybrid models  
- Personalization  
- Exploration vs exploitation  

---

## 📈 Metrics

- CTR (Click Through Rate)  
- Precision / Recall  
- Watch time / engagement  

---

## 💬 Sample Answer (Short)

“A recommendation system suggests items based on user behavior.  
It uses techniques like collaborative filtering and content-based filtering to improve personalization and engagement.”
